# Python-Log-Analyzer
A Python script that parses system log files and counts occurrences of key log levels (ERROR, WARNING, INFO) to assist with troubleshooting.

## Features
- Reads and processes log files
- Categorizes log entries by severity
- Outputs count of ERROR, WARNING, and INFO messages

## Technologies
- Python
- Tested in Linux environment

## Usage
python3 analyzer.py
Enter log file name: system.log

## Purpose:
Built to simulate basic log analysis tasks used in IT support and system administration.
